CT Image Reconstruction
from DICOM Data

MATLAB implementation of the Radon transform and Fourier Slice Theorem to reconstruct a 2D image from 23,000+ sinogram data points — a numerical inverse problem requiring multi-step Fourier analysis and interpolation.

Language MATLAB
Data DICOM medical image (128×128)
Context Independent · USC Biomedical Engineering
Overview

CT scanners do not directly capture a 2D image. They measure a set of 1D line-integral projections of the object at different angles — the sinogram. Recovering the original image from these projections is the core inverse problem of tomography, solved here via the Fourier Slice Theorem: each 1D projection corresponds to a radial slice through the 2D Fourier transform of the image.

The pipeline takes a DICOM medical image, constructs the sinogram by rotating and projecting the image at 180 angles, assembles the full 2D Fourier transform from the 1D projection slices via polar-to-Cartesian interpolation, then inverts it to recover the reconstruction.

Connection to volatility calibration

Both this problem and volatility surface calibration are ill-posed inverse problems in Hadamard's sense. In CT, the ramp filter required for filtered back-projection amplifies high-frequency noise — exactly as numerical differentiation of ∂²C/∂K² amplifies noise in Dupire's formula. Both require regularization to be stable. This structural analogy is the central narrative of the stochastic vol project.

Pipeline
01
Load & crop DICOM image
Read the raw DICOM file and crop a 128×128 region of interest. This reduces computation while preserving the key structure to reconstruct.
02
Construct the sinogram
Rotate the image at each of 180 integer angles (0°–179°) and sum along columns to produce 1D projections. Stacking these gives the sinogram — a (180 × 128) matrix where each row is one angular projection.
03
1D FFT of each projection
Apply the 1D FFT to each projection. By the Fourier Slice Theorem, the result is a radial slice through the 2D Fourier transform of the original image at the corresponding angle.
04
Assemble 2D Fourier transform via interpolation
The 1D FFT slices are in polar coordinates (angle, frequency). Convert to Cartesian (x, y) using pol2cart, then interpolate onto a regular grid using griddata to assemble the full 2D Fourier transform.
05
Validate against direct 2D FFT
Compare the assembled Fourier transform against a direct fft2 of the original image. Agreement in log-magnitude confirms the Fourier Slice Theorem is being applied correctly.
06
Inverse FFT → reconstructed image
Apply ifft2 to the assembled transform and take the absolute value. The output is the reconstructed image — recovered purely from the 1D projections without any direct image data.

Code — analysis.m

The full pipeline in a single MATLAB script. Section markers (‍%%) correspond directly to the pipeline stages above.

analysis.m MATLAB
1clear, clc, close all
2
3%% Part 1
4
5dicomImage = dicomread("image10.dcm");
6
7figure
8imshow(dicomImage, [])
9title('Original DICOM Image')
10
11croppedImage = imcrop(dicomImage,[175 125 127 127]); % Crop 128x128 section
12
13figure
14imshow(croppedImage, [])
15title('Cropped 128x128 Section')
16
17%% Part 2
18
19numAngles = 180;
20projections = zeros(numAngles, size(croppedImage, 1));
21
22for angle = 1:180
23 rotatedImage = imrotate(croppedImage, angle, 'bilinear', 'crop');
24 projections(angle, :) = sum(rotatedImage, 1);
25end
26
27figure
28imagesc(projections)
29colormap('gray')
30xlabel('Detector Pixels')
31ylabel('Projection Angle (degrees)')
32title('180 1D Projections (Sinogram)')
33
34%% Part 3
35
36pad = padarray(croppedImage,[50 50],'both');
37
38for i = 1:180
39 rotate = imrotate(pad,(i-1),'bilinear','crop');
40 sinogram(:,i) = sum(rotate);
41end
42
43dfft = fftshift(fft(fftshift(sinogram)));
44
45figure
46imshow(log(abs(dfft)),[])
47
48%% Part 4 and 5
49
50theta = repmat((0:1:(180-1))*pi/180, length(dfft), 1);
51bound = (length(dfft)-1)/2;
52rho = repmat((-ceil(bound):floor(bound))', 1, 180);
53
54[x, y] = pol2cart(theta, rho);
55[XI, YI] = meshgrid(-ceil(bound):floor(bound));
56
57assembled_dfft = griddata(x, y, dfft, XI, YI, 'linear');
58assembled_dfft(isnan(assembled_dfft)) = 0;
59
60figure
61imshow(log(abs(assembled_dfft)),[])
62colormap('gray')
63title('Interpolated 2D Fourier Transform')
64
65%% Part 6
66
67direct = fft2(croppedImage);
68
69figure
70imshow(log(abs(fftshift(direct))),[])
71colormap('gray')
72title('2D Fourier Transform (Magnitude)')
73
74%% Part 8
75
76inverseImage = abs(fftshift(ifft2(fftshift(assembled_dfft))))' ;
77
78figure
79imagesc(inverseImage)
80colormap('gray')
81title('Reconstructed Image from Inverse 2D Fourier Transform')
